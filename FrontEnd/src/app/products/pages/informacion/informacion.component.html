Información de la Web / Proyecto

El proyecto que estamos desarrollando es un sistema de gestión de productos de indumentaria que facilita la administración y visualización de un inventario en línea.

Backend

El backend está construido en Python usando FastAPI y se conecta a una base de datos MongoDB. Este componente se encarga de manejar las operaciones de creación, lectura, actualización y eliminación (CRUD) de productos, que en este caso representan artículos de ropa. Cada producto tiene las siguientes propiedades:
  • id: Identificador único del producto.
  • nombre: Nombre del producto.
  • precio: Precio del producto, almacenado como un valor entero.
  • descripcion: Descripción del producto, que proporciona detalles adicionales.
  • disponible: Indica la disponibilidad del producto.
  • talle: Talle del producto.
  • marca: Marca del producto.
Estas funciones permiten realizar las operaciones necesarias para gestionar el inventario: almacenar un nuevo producto, obtener el listado completo de productos, buscar un producto específico por su id, editar las propiedades de un producto y eliminarlo de la base de datos.

Frontend

El frontend, desarrollado en Angular y utilizando componentes de PrimeNG, consume el backend para mostrar los productos en una interfaz amigable para el usuario. Los productos se muestran en una tabla que permite la visualización y organización del inventario. Además, la interfaz incluye un formulario que permite agregar productos al inventario de forma sencilla. Cada vez que se agrega un producto, este se guarda en un carrito temporal, permitiendo añadir varios productos antes de guardarlos en la base de datos. La aplicación también permite al usuario editar o eliminar productos individuales desde la misma interfaz. Este sistema proporciona una solución completa para gestionar productos de indumentaria, facilitando la administración de un inventario de manera eficiente y accesible.
